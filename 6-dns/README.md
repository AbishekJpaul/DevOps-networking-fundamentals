# DNS Resolution

DNS converts domain names into IP addresses.

Example:

google.com -> 142.250.x.x

Process:

1. Browser sends DNS request.
2. DNS server returns the IP address.
3. Browser connects to the destination server.

---

# TCP 3-Way Handshake

TCP establishes a reliable connection using three steps:

1. SYN
2. SYN-ACK
3. ACK

Client                     Server

SYN -------------------->

      <------------------ SYN-ACK

ACK -------------------->

Once the handshake is complete, data transmission begins.
